## 二、遇到的问题与解决

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 类别筛选无效 | 筛选逻辑未正确连接 Store | 重构筛选器与状态管理连接 |
| 角色创建按钮禁用 | 前端临时调试代码未移除 | 移除禁用逻辑，启用按钮 |
| 角色-技能不关联 | 配置写入与读取逻辑不一致 | 统一关联数据的读写方法 |
| MD格式解析失败 | Frontmatter 字段顺序不固定 | 固定字段顺序，实现自动格式化 |
| 文件选择器无响应 | 浏览器用户手势验证拦截 | 改用 input 标签方案替代 |
